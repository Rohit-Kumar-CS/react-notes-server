# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

# 📝 Paste App

A modern and responsive Paste Management application built with **React**, **Redux Toolkit**, **React Router**, and **Tailwind CSS**. Users can create, edit, view, delete, copy, and share text snippets with ease.

---

## 🚀 Features

- ✨ Create new pastes
- ✏️ Edit existing pastes
- 👀 View paste details
- 🗑️ Delete pastes
- 📋 Copy paste content to clipboard
- 🔗 Share paste links
- 💾 Local Storage support
- 🔍 Search pastes
- 📱 Fully responsive UI
- 🎨 Modern gradient design

---

## 🛠️ Tech Stack

- React.js
- Vite
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- React Hot Toast

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/paste-app.git
```

Go to the project directory

```bash
cd paste-app
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

---

## ⚙️ Dependencies

```bash
npm install react-router-dom
npm install @reduxjs/toolkit react-redux
npm install react-hot-toast
npm install tailwindcss
```
